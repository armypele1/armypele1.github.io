About Me

Recent computer science graduate from Durham University, looking to enter a role where I can learn from experienced team members. Passionate about learning and applying new technologies in the gaming industry.

Featured Projects

Adaptive Educational Game In VR

This project presents the design, implementation, and evaluation of a novel VR educational game framework that incorporates an adaptive questioning system based on a new interpretation of the Elo-rating system and user-initialized customization through an in-game shop.

View Project

Generative Deep Learning Model

Here I present image generation using a denoising diffusion probabilistic model which utilizes classifier-free diffusion guidance. The model is trained on the CIFAR-10 dataset at a resolution of 32x32, for a total of around 1 million iterations.

View Project

VR Graphics Engine (almost) From Scratch

In this project I extend functionality of a rudimentary 3D graphics engine to include basic rendering, tracking, physics, and distortion pre-correction for VR. I limited my use of external libraries, which lead to me writing my own classes for vectors, matrices, and quaternions.

View Project

Song Recommender System

Here I develop a system that recommends new, unheard music to users that they will react to most positively based upon the music they have previously listened to. I utilize the Million Song Dataset (MSD) for this purpose, accessed via an AWS EC2 instance. A state-of-the-art model geared towards sparse data is used to achieve accurate personalised recommendations.

View Project

First-Person 3D Platformer

This project shows the design and development of a mario-64-esque platformer with a first-person shooter twist. The aim here was to demonstrate my ability to create all aspects of a complete video game, including menus, HUD, game physics and accessibility features.
